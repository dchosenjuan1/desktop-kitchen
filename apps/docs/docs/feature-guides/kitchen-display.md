# Kitchen Display

The Kitchen Display System (KDS) shows incoming orders in real-time for kitchen and bar staff.

## Setup

1. Open the POS on a kitchen-dedicated tablet or monitor
2. Log in with a **Kitchen** or **Bar** role PIN
3. The display automatically shows the appropriate order queue

## How It Works

### Order Flow

When a POS cashier confirms an order:

1. The order appears on the kitchen display instantly
2. Each item is shown with its modifiers and special notes
3. Kitchen staff taps an item when they start **preparing** it
4. When all items are ready, the order moves to **ready** status

### Order Cards

Each order card shows:
- **Order number** (e.g., #0042)
- **Order source** (POS, Uber Eats, Rappi, DiDi Food)
- **Time elapsed** since the order was placed
- **Items** with modifiers and notes
- **Priority indicators** for time-sensitive orders

### Status Progression

```
New Order → Preparing → Ready → Completed
```

Tap an order card to advance its status. Color coding helps identify urgency:
- **White**: Just arrived
- **Yellow**: Approaching target time
- **Red**: Overdue

## Bar Display

Staff logged in with the **Bar** role see only drink items from orders. This allows drink preparation to happen in parallel with food preparation.

## Tips

- Mount the display at eye level for kitchen staff
- Use a screen large enough to show 4-6 orders simultaneously
- Consider a dedicated tablet that stays logged in as Kitchen role
