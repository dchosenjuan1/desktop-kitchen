# Reports & Financial Targets

Access sales analytics, financial reports, and set performance targets.

## Sales Reports

### Daily Summary
- Total revenue (MXN)
- Number of orders
- Average order value
- Payment method breakdown (cash vs card)
- Order source breakdown (POS vs delivery platforms)

### Time-Based Analysis
- Hourly sales heatmap — identify peak and slow periods
- Daily, weekly, and monthly trends
- Year-over-year comparison

### Item Performance
- Best-selling items by quantity and revenue
- Worst performers and candidates for removal
- Category-level breakdown
- Modifier popularity (which customizations are most common)

## Financial Targets

Set targets and track progress:

1. Go to **Reports** > **Targets**
2. Set daily, weekly, or monthly revenue targets
3. The dashboard shows real-time progress toward targets
4. Color indicators show if you're on track (green), falling behind (yellow), or missing targets (red)

## Delivery Reports

Delivery-specific metrics are covered in [Delivery Intelligence](./delivery-intelligence):
- Per-platform revenue and commission
- Net profit after platform fees
- POS vs delivery revenue mix

## Employee Performance

- Orders processed per employee
- Average transaction value by employee
- Shift-level reporting

## Inventory Reports

- Stock value (current inventory valued at cost)
- Consumption rates
- Waste and shrinkage tracking
- Purchase order history

## Exporting Data

Reports can be viewed on-screen with interactive charts powered by Recharts. For data export, use the API endpoints to pull raw data.

## Charts & Visualization

The reporting dashboard uses interactive charts:
- **Line charts**: Revenue trends over time
- **Bar charts**: Category and item comparisons
- **Pie charts**: Payment method and order source distribution
- **Heatmaps**: Hourly sales patterns
